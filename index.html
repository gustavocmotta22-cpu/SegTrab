2. Política de Privacidade (LGPD)

Em conformidade com a Lei 13.709/2018 (LGPD), informamos:

📱 Dados coletados

Nome completo, e-mail, CPF, telefone, cargo e empresa. Também são armazenados os relatos, planos de ação e documentos que você inserir no aplicativo.

🔒 Como são armazenados

Apenas no seu dispositivo (localStorage do navegador). Não há servidor externo, banco de dados em nuvem nem transmissão de dados pessoais a terceiros. Os dados existem somente no navegador utilizado.

📊 Pesquisa de satisfação

Ao responder a pesquisa final, você tem a opção de enviar suas respostas por e-mail ao desenvolvedor. Este envio é opcional, voluntário e ocorre através do cliente de e-mail do seu próprio dispositivo. Nenhum dado é enviado automaticamente.

🗑️ Como excluir seus dados

Para excluir todos os seus dados: acesse as configurações do seu navegador → Privacidade → Limpar dados de navegação → Dados de sites. Isso remove todos os dados do SegTrab do seu dispositivo.
